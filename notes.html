Important Concepts

World Wide Web

A collection of html pages located on a remote server.
These can be accessed through a computer's browser

HTML

HTML stands for Hypertext Markup Language. Most of the pages on internet use HTML. This page summarizes the basic concepts.

HTML Tags and Elements

Elements tell a browser how to display content on a web page.
Tags are used to tell the browsers the type of such elements.

Inline and Block Elements

HTML elements can be inline or block. Inline elements are inline (duh) while block elements form an "invisible" box around the content that's inside them.
